API Documentation


Bingo

v 1.0

The API for our bingo game. The API is set up so, that the CORS allows for the use of it outside of our local network.
Root: api.morecis.co/v1/bingo/

Bingo card items
[GET] /items
Get the items for the bingo card in .json form.
JavaScript example

$.getJSON("https://api.morecis.co/v1/bingo/items")
